
# SAPX Design Portfolio

A premium, motion-led portfolio website for a multidisciplinary design studio, built to present services, projects, testimonials, and client trust signals with a clean and modern visual language.

## Project Snapshot

- Design-first studio website with strong visual storytelling
- Sections include Services, About, Projects, Testimonials, and Contact
- Interactive project and service detail experiences
- Client logo marquee for social proof

## Recent Updates

- Added Mata Chanan Devi Hospital to the client logo system and marquee
- Expanded Product Design capabilities with IoT
- Updated architecture service capabilities to include healthcare and industrial focus areas
- Refined project image sizing guidance for better visual balance across cards and modal views
- Updated mid-page CTA copy to: "Book a free call with our experts. Let's talk."

## Notes for Client Review

- Content updates can be quickly managed in component data blocks
- Project images are controlled per project entry in the Projects section
- CTA currently scrolls users to the Contact section for inquiry submission

## Local Preview

1. Install dependencies: `npm install`
2. Start development server: `npm run dev`
